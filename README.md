
## 简介
```
vue3.0后台简易模版。

配置了axios、element、vuex、less、vue-router。二次封装了element的table组件及form表单组件。

能够快速搭建一个后台管理系统。

推荐使用yarn管理依赖

npm i yarn -g

yarn config set registry htts://registry.npm.taobao.org
```

## 安装依赖
```
yarn
```

### 运行开发环境
```
yarn serve
```

### vue 打包
```
yarn build
```

### vue 运行 自定义开发环境
```
yarn serve -test
```
